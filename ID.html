Cuándo usar una clase o un identificador

Se utiliza un id cuando el estilo debe aplicarse a un único elemento en la página, ya que el identificador debe ser único dentro del documento HTML.
Se usa una clase cuando el estilo puede aplicarse a varios elementos, permitiendo reutilizar los mismos estilos en diferentes partes de la página.

Diferencias en uso y especificidad

Un id tiene mayor especificidad que una clase. Esto significa que, en caso de conflicto, los estilos definidos por un id prevalecerán sobre los de una clase.
Solo un elemento puede tener un id particular, mientras que varias etiquetas pueden compartir una misma clase.
