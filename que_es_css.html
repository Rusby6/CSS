¿QUE ES HTML Y PARA QUE SIRVE?

CSS es un lenguaje usado para describir la presentación visual de un documento escrito en HTML. Mientras que HTML se encarga de la estructura y el contenido de una página web, CSS define cómo ese contenido debe ser mostrado al usuario. En resumen, CSS basicamente sirve para "decorar" nuestra pagina web, "pintarla"
