¿QUE ES HTML Y PARA QUE SIRVE?

CSS es un lenguaje usado para describir la presentación visual de un documento escrito en HTML. Mientras que HTML se encarga de la estructura y el contenido de una página web, CSS define cómo ese contenido debe ser mostrado al usuario.
En resumen, CSS basicamente sirve para "decorar" nuestra pagina web, "pintarla"

FORMAS DE APLICAR CSS EN HTML

Como dije anteriormente, CSS se puede aplicar a HTML, y estas son algunas de ellas:
-El CSS en línea (inline CSS) es una forma de aplicar estilos directamente a un elemento HTML, usando el atributo "style" dentro de la etiqueta del elemento.
Este método aplica los estilos solo a ese elemento específico, sin afectar a otros en la página.
-El CSS en el encabezado o CSS interno se utiliza dentro de la etiqueta "style" en la sección de un documento HTML.
Este método te permite aplicar estilos a múltiples elementos en la página sin tener que escribir el estilo directamente en cada etiqueta HTML (como en el caso de CSS en línea).
Es útil cuando deseas agregar estilos específicos a una sola página sin crear un archivo CSS externo.
