-El margen es el espacio exterior que rodea un elemento. Controla la distancia entre el borde de un elemento y los elementos circundantes.
-El padding en CSS se refiere al espacio interno entre el contenido de un elemento (como el texto o una imagen) y su borde. En otras palabras,
el padding crea un área de separación entre el contenido y el borde del elemento, pero sigue siendo parte del área del elemento, por lo que se aplica dentro de su borde.
-La propiedad border define el borde alrededor de un elemento HTML. Este borde puede variar en grosor, estilo y color.
-Las propiedades width y height se utilizan para definir el ancho y la altura de un elemento. Estas propiedades son esenciales para controlar el tamaño de cajas o contenedores
